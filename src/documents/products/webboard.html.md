## webBoard

Simple WhiteBoard Using WebSocket.
It helps your team to have a discussion or brainstorming at online.

### DEMO

[visit demo](http://webboard.jit.su)

### Getting Started

  ``` bash
  git clone git://github.com/georgeOsdDev/webBoard.git
  cd webBoard
  npm install
  node app.js```

### Technique
* Server-side
  * coffee-script
  * express & ejs
  * stylus & nib
  * socket.IO
* Client-side
  * jquery
  * jquery-UI
  * socket.IO-client
* Host
  * nodejitsu
